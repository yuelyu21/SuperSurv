# METABRIC Breast Cancer Dataset

A subset of the Molecular Taxonomy of Breast Cancer International
Consortium (METABRIC) dataset to demonstrate the SuperSurv package.

## Usage

``` r
metabric
```

## Format

A data frame with the clinical and genomic variables:

- duration:

  Survival time.

- event:

  Event indicator (1 = event, 0 = censored).

- x0:

  Feature x0.

- x1:

  Feature x1.

- x2:

  Feature x2.

- x3:

  Feature x3.

- x4:

  Feature x4.

- x5:

  Feature x5.

- x6:

  Feature x6.

- x7:

  Feature x7.

- x8:

  Feature x8.
